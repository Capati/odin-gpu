# Info

Print current version and selected adapter information.

```shell
build.bat|sh info run
```

Example output:

```shell
Version: 0.0.0

Available adapter(s):

NVIDIA GeForce RTX 3060
  - Driver: 32.0.15.8180
  - Type: Discrete GPU with separate CPU/GPU memory
  - Backend: Direct3D-11
Microsoft Basic Render Driver
  - Driver: 10.0.22621.4249
  - Type: Cpu / Software Rendering
  - Backend: Direct3D-11

Selected adapter:

NVIDIA GeForce RTX 3060
  - Driver: 32.0.15.8180
  - Type: Discrete GPU with separate CPU/GPU memory
  - Backend: Direct3D-11
```
